# 📚 BookApp

تطبيق Flutter لعرض الكتب مع تسجيل دخول وتسجيل مستخدم باستخدام Dio و SharedPreferences.  
تم بناء التطبيق بطريقة منظمة باستخدام Repository Pattern، وكل البيانات تأتي من API بدون أي هاردكود.
